# **dwm**
My build of dwm - dynamic window manager

**Added patches:**

* fullgaps
* alpha
* pertag-perseltag
* centeredmaster
* fibonacci
* gridmode
* alternativetags
* swallow
* scratchpad
* cyclelayouts

**Using:**

* st
* dmenu
* rofi (alt+shift+p)
* slock

**Dependencies:**

* libxcb (swallow)
* Xlib-libxcb (swallow)
* xcb-res (swallow)
* font awesome (alternativetags)
